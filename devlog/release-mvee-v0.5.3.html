v0.5.3 — Stability Patch

This patch resolves the board-reported bugs from our April 20 playtest. Animals now spawn deterministically across biomes instead of repeating the same set every game. Angel NPCs handle LLM failures gracefully with fallback messages, and the prompt leak detection has been hardened.

The Join Universe flow no longer creates inconsistent state when entering through the universe browser. Species are displayed by their canonical names, and the LLM proxy now has server-side rate limiting with proper 429 retry logic.

On the lore side, we continued the IRL mythology name remediation and aligned the Alfar species profile with the Akashic Records canon. Ten fixes total across spawning, rendering, networking, and lore consistency.
