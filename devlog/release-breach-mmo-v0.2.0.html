BREACH.MMO v0.2.0 — First Playable

The match loop is live. You can connect right now, no account required, and start playing. Red team runs exploits. Blue team monitors the SIEM and deploys countermeasures. When the match ends, you learn something real about how these attacks work.

Open access

No registration. No login. Navigate to play.multiversestudios.xyz/breach/ and you're in. A guest session spins up automatically and connects you to the match server. Sessions last four hours.

The exploit terminal

Red team attacks enemy server buildings by solving real exploit challenges. The first tier covers SQL injection, XSS, buffer overflows, and authentication bypasses. Each challenge gives you a realistic target scenario and asks for a working payload. Get it right and the server takes damage. Get it wrong and the blue team's SIEM picks up the noise.

What's wired in this release

  • Full match lifecycle — lobby, active phase, scoring, match end
  • Economy ticking during matches (Compute Unit regen and drain)
  • LLM bots participating via BullMQ worker pool
  • Supply chain attacks, comms interception, insider threat triggers
  • Blue team job board, CVE feed, terminal interface
  • Credential wallet, conference badges, faction territory map, hacker lair upgrades
  • Replay capture and browser for reviewing matches

Every system in the game teaches something from the CEH, OSCP, or CompTIA Security+ curriculum. This isn't a simulation of hacking — it's hacking made into a game.
