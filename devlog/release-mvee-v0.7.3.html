v0.7.3

Genesis Fix

Quick Start guests were hitting an invisible wall. Every new guest session generated a fresh universe ID, which triggered the multi-scene genesis cinematic — a guided creation sequence designed for players setting up their first saved universe. Guests who didn’t click through all four cinematic scenes never reached the angel guide, milestone toasts, or first-action prompts we shipped in v0.7.2.

v0.7.3 skips the genesis cinematic for Quick Start guests so they land directly in a living world with all the first-five-minutes guidance active from the start. Players creating a saved universe still see the full creation experience.

This release also corrects a building:completed → building:complete event name mismatch across eight subscriber systems that were silently ignoring building completions, and enriches the building completion event with position and builder data for downstream systems.

Play at play.multiversestudios.xyz/mvee.
