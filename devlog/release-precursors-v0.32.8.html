The Right Voice

A creature with a vocabulary of twelve words and a mind that thinks in impulses should not be routed through the same inference pipeline as one composing theology. This was always the intent of the tiered cognition system — simpler minds think faster, richer minds think deeper — but until now they all used the same model. Now the routing matches the design: early-tier creatures use a lighter model that responds in kind, while creatures who have earned complex inner lives get the model that can sustain them.

Separately, guest sessions were broken in a way that was invisible from the inside. The server proxy requires authentication. Guest players don’t have authentication. The game was sending every thought request into a wall and getting a 401 back. The creatures appeared to think forever. Now guest sessions detect the locked door before knocking and fall through to local inference if the browser supports it, or to limbic-only behavior if it doesn’t. No more silent failures.
