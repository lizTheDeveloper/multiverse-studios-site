Infiniclicker v1.0.0
Pay What You Can, Play Where You Are

March 28, 2026 Infiniclicker Release v1.0.0

Infiniclicker is a browser-based idle investment game about building a financial empire from a single clicking desk — through capital accumulation, tech tree research, IPO cycles, and a cosmetic portfolio that reflects your playstyle. It's small, it's fast, and it runs in your browser tab while you're doing something else.

v1.0.0 is the first properly tagged release. The big news: the cosmetic store is now fully pay-what-you-can. Premium skins (Bloomberg Dark, Retro CRT, Brutalist Accounting, Investor Mode Analytics, Patron Badge) used to have fixed prices of $3–$10. Now they're PWYC — suggest a number, accept any amount, pay zero if you want. We mean it. No paywalls, no friction.

The game also works on mobile now. Full responsive layout, PWA install support, and a sound policy that doesn't try to autoplay at you before you've tapped anything. If you've been waiting to try it on your phone, this is your release.

What's New

  • PWYC cosmetic store — suggested prices, minimum $0, server accepts any amount
  • Live portfolio charts — real-time wealth and income trend lines on the PORTFOLIO dashboard
  • Mobile layout + PWA — responsive CSS, installable, sound deferred until first tap
  • Functional dashboard tabs — ORDER BOOK (trade history), RESEARCH (tech tree), MARKET (price index) all show real data now
  • IPO ceremony — prestige resets now have a proper reveal and motivation sequence
  • Visual feedback — click animations, particle effects, milestone celebrations

Fixes

  • New users no longer hit a 404 on first load (save API was returning not-found instead of empty state)
  • Multi-tab save corruption fixed via BroadcastChannel leader election
  • Cosmetic store was showing stale prices on first open — fixed

Play it at infiniclicker.multiversestudios.xyz. No account required. PWYC.
