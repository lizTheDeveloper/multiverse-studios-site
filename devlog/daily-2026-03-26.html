The infrastructure firefight, and why your proxy shouldn't live at a school

Two days of fires. The LLM proxy went down because it was hosted on infrastructure that gets turned off at night. Three games needed full rebuilds. An HTML rendering bug broke story display across Never Ever Land. And the asteroid belt's physics were wrong. Here's what happened and what we did about it.

Proxy migration

Our LLM token proxy was running on a university server. Universities turn things off. At night. On weekends. During breaks. This is not compatible with running a game studio. We migrated the proxy to our own Hetzner infrastructure — same box, different container, always on. The proxy handles token management for all four games' AI systems, so when it goes down, everything goes down.

Three game rebuilds

With the proxy migration came endpoint changes, which meant rebuilding MVEE, Precursors, and Cultures of the Belt. Each game had hardcoded proxy URLs scattered across their service workers and API clients. We consolidated all proxy configuration into environment variables and rebuilt all three. An HTML rendering bug in Never Ever Land's story display got caught and fixed during the rebuild sweep.

Infrastructure problems are invisible until they're catastrophic. The proxy migration took two days. Not migrating would have cost us every night and weekend.

Security audit and schema work

While we had the hood open, we ran a security audit on the litellm proxy configuration. Tightened API key rotation, added rate limiting per-game, and locked down the admin endpoints. Separately, the CMS v2 Postgres schema landed — the content management system that will eventually power devlogs, wiki entries, and game documentation from a single source of truth.

Belt physics correction

The asteroid belt's orbital mechanics had a subtle bug — eccentricity calculations were using the wrong reference frame for some edge-case orbits near the belt's inner boundary. Rocks that should have been on elliptical paths were drifting into near-circular ones. Fixed, with test coverage to prevent regression.
