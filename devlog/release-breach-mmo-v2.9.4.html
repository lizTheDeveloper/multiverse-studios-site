BREACH.MMO v2.9.4 — 2026-06-18

Game World Restored

v2.9.4 is a reliability patch. The game world canvas was rendering blank for all players due to sprite atlas files missing from the production build — a deployment gap that crept in during a recent infrastructure change. All six atlas files are now committed and serving correctly. If you loaded the game and saw an empty black canvas over the last day or two, this is the fix.

Sprite atlases restored — terrain, characters, buildings, UI elements, bots, and effects all render correctly again. The red-team and blue-team game world is fully visible. No gameplay data was lost; this was purely a visual break.

Checkout source tracking — we added conversion source tagging to checkout sessions so we can see which in-game CTA actually drives Analyst Pass purchases. This is internal instrumentation; nothing changes for players. The data helps us improve the upgrade experience over time.
