v0.5.5 — Smarter Agents

This release makes village agents significantly more resilient. When the LLM returns malformed responses, agents now retry with a simplified parser before falling back to productive behavior instead of getting stuck. The rethink interval for idle agents dropped from five minutes to about one minute, so settlements feel noticeably more alive.

Fallback behavior got a major upgrade too. When the cognition pipeline is unavailable, agents now prioritize farming, building, and food-seeking over aimless wandering. Building costs were also synced with the blueprint registry to prevent false "insufficient resources" messages.

On the world-generation side, deity personalities now require seriousness and compassion as core traits, giving each universe a more consistent divine character from the start.
