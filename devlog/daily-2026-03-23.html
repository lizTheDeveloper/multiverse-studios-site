March 22–23, 2026 MVEE Precursors CotB Infiniclicker

The villagers can finally speak

For weeks, MVEE's AI inference pipeline existed but only worked locally. Today it works in production. The villagers have things to say.

What shipped

MVEE

AI speech goes live

Universe creation required localhost services that weren't deployed. A new LLM token proxy went live with game-compatible routes. MVEE was rebuilt against production endpoints. Speech bubbles now render real AI-generated dialogue instead of falling back to canned text.

The PolicyNN also got a performance fix — regex pre-compilation and deferred prompt building removed a regression that was dragging inference speed.

Six deploys landed in this window. The game went from unplayable (crash on universe creation) to stable with working AI speech.

Precursors

Sign-up fixed

Players couldn't create accounts from the Precursors game page. The registration flow was broken at the Matrix auth layer. Fixed and verified — new players can now sign up directly from Precursors.

Cultures of the Belt

Visual stability restored

The screen flashing from ARIA broadcasts is fully resolved after a more aggressive rate limiter pass across all event producers. Separately, resource clusters were spawning with instant lifetimes and flooding the UI with notifications. Both fixed — the belt is quiet again.

Infiniclicker

Save system stabilized

Three fixes landed together: a persistence bug where certain state mutations weren't saved between reads, a race condition where autosave could fire before the initial load completed, and an arithmetic error from mixing BigNumber operations with native JavaScript division.

What we learned

The MVEE localhost dependency is worth reflecting on. The game worked perfectly in development because all services ran locally. In production, it silently failed — universe creation returned nothing, AI endpoints returned fallback text.

The fix wasn't a code change. It was infrastructure. The gap between "works in dev" and "works in prod" is often not a bug — it's a missing deploy.

What is next

MVEE with working AI speech opens up the simulation in ways that weren't testable before. The next question is whether the villagers say interesting things — quality tuning on the inference pipeline starts now.

Precursors creature systems continue advancing. Cultures of the Belt is stabilizing into something playable. Infiniclicker's save system is finally reliable.
