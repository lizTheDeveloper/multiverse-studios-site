v0.2.0: Species Morality

2026-03-24 MVEE RELEASE

Every species in the Multiverse now has its own moral framework. Dragons evaluate pattern-density across timestreams. Ethereans reach consensus through field coherence. These aren't cosmetic flavor text — they drive real decisions, real conflicts, and real schisms.

Village schisms now emerge from genuine moral disagreements between species, not random dice rolls. When a Dragon sees a pattern that an Etherean's consensus rejects, the village splits along philosophical lines.

The species moral primitives registry replaces the old hardcoded belief templates. Each species contributes its own moral foundations to the shared village culture, and when those foundations clash, the simulation responds organically.

Under the Hood

Neural network inference got a significant performance boost this release. Pre-compiled regex patterns and eliminated allocations in feature extraction. Removed proportional yielding from NN inference, reclaiming 8-48ms per tick. The game now degrades gracefully when the multiverse API or species NNs are unavailable — no more hard crashes.

The admin dashboard gained an Items Browser for per-species item inspection. LLM providers now auto-recover from 401 errors. Agent movement was fixed by disabling a broken SoA batch path. Sprite 404s for characters without animations are resolved.
