BREACH.MMO v2.4 — 2026-06-09

Season Engine

v2.4 completes the competitive layer that started with crew matches in v2.3. Seasons now run on a 13-week cron schedule — no manual admin rollover needed. Every season closes with a permanent title badge awarded to the top three players per faction. These badges don’t reset. They accumulate. “Vanguard — Season 1” means something different two years from now than it does today.

Faction War Season Events add a two-week competitive window inside each season. Curators map these to real APT campaigns — an “Operation Midnight Blizzard” event awards Red team 2× season points for exfil challenges while the event is active. The event banner shows in-game. The Breaking CVE alert toast now displays the CVSS score and attack vector inline so the urgency signal is immediate: CRITICAL | RCE | Network before you decide whether to click.

The community mission library gains Mission Collections — named groups of missions curated into a series. Collections show your completion state across the set, so a well-designed multi-part training arc has a visible finish line. The backend shipped in v2.3; the browsable UI is live now.
