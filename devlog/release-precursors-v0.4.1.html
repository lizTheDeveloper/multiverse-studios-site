Precursors v0.4.1 — Silence the Babble

March 21, 2026 Precursors Patch

Quick patch to fix an audio annoyance. The VoiceSystem had an idle babbling loop that spoke gibberish syllables ("nnn", "rrr", "mnn") aloud via the browser's SpeechSynthesis API every 30-90 seconds per norn. If you had a few norns alive, your speakers became a chorus of robot nonsense.

That loop is now disabled. Event-driven speech — when norns actually have something to say via the LLM — still works. A proper pre-generated audio system for Nornish utterances is in the pipeline.

We also added vocabulary hints to the LLM prompt so that when norns do speak, their words better match established Nornish vocabulary rather than generating random syllables.

Play the quieter version at multiversestudios.xyz.
