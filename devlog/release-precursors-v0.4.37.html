Precursors v0.4.37: Cultural Exchange

March 25, 2026 Precursors

The big addition in this patch is cross-species music teaching. Ettins can now learn songs from Norns — when an Ettin hears a Norn singing nearby, they pick up the melody and begin singing it themselves. This is the first step toward the emergent cultural exchange system where species share and transform each other's musical traditions.

The Shee Ark has been completely redesigned. The old rocket-ship sprite has been replaced with a proper multi-biome generation ship that hints at the scale and purpose of the vessel. It finally looks like something a civilization would build to carry life across the stars.

On the stability side: the bug reporter now uses Phaser's native snapshot API for accurate screenshots, and a warmup guard prevents wave unlocks from firing the instant the game loads.
