BREACH.MMO v1.8.0 — 2026-06-03

Education Depth Update

v1.8.0 is the certification prep update. Two new learning paths take players from zero security background all the way to PNPT-level Active Directory attacks, and the game now speaks the language of real CTF exams.

Learning Paths — the eJPT Fundamentals Path is a 12-mission beginner on-ramp: no prior security knowledge required. NetworkFundamentals, HttpIntercept, MetasploitBasics, and SimpleInjection challenges walk players through the foundations, with 0.5× CU cost on the first three missions to keep the barrier low. Complete the path and earn the "eJPT Ready" badge. The PNPT Internal Network Pentest Path is the advanced counterpart: 16 missions covering AD attacks — LLMNR Poisoning, Kerberoasting, DCSync, and Pivoting — culminating in a 4-hour practical exam against a 5-machine isolated network. Real exam pressure, in-game.

CTF Flag Submission — type submit flag{'{...}'} in the terminal to validate a flag server-side. The format will be familiar to anyone who has competed in a real CTF. Your profile now tracks a flag counter, and you can toggle between card mode (guided) and CTF mode (raw terminal) depending on how much hand-holding you want.

OSCP Writeup Replay — after completing any OSCP-aligned challenge, a panel opens comparing your steps against the expert kill chain. Twelve static writeup scenarios cover the most common OSCP PEN-200 attack paths. It is a debrief, not a punishment — dismiss it permanently once you have internalized the lesson.
