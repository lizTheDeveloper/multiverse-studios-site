Precursors v0.2.1 — Mobile, Sessions, Thinking

This is a patch release focused on things that were quietly broken for a large number of players. No new species. No new systems. Just a world that works better for more people.

Mobile gameplay is fixed

About half our players are on phones. In v0.2.0 they were getting a letterboxed canvas — the game rendering into a tiny 375×211px box with black space filling the rest of the screen. That's 16% of the available area. The game was effectively unplayable on mobile.

It's fixed. The canvas now scales to fill the viewport, and the chat interface — which had collapsed the input field and buttons to an unusable width — has been corrected as well. If you've tried Precursors on your phone and given up, try again.

Norns can think again

The GROQ API key on the production server had expired. Norns were generating 401 errors instead of thoughts. This is the key that drives LLM cognition — the ability of Norns to interpret their biochemical state, form sentences, and respond to the world with actual language rather than placeholder text.

The key has been rotated and verified. Norns are thinking again.

Login once, play everything

A session cookie was set to SameSite=Strict, which meant players had to log in separately on each game page. That's now Lax. If you're signed in at play.multiversestudios.xyz, you stay signed in across Precursors, MVEE, and everything else on the platform.

Play it now at play.multiversestudios.xyz. Free. In your browser. Bring your own Groq API key if you want full LLM cognition, or watch the biochemistry run on its own.
